常用图片效果

圆角图片

使用 border-radius 属性可以实现圆角（圆形）图片。

Thumbnail图片

使用 border 属性可以实现“拇指图”。

响应式图片

响应式图片会自动调整自身大小以适应屏幕尺寸

半透明图片

图片滤镜

属性值 描述
h-shadow 必须，指定水平方向上的阴影距离，具体像素值，负值会使阴影在图像左边
v-shadow 必须，指定垂直方向上的阴影距离，具体像素值，负值会使阴影在图像上边
blur 可选，必须为具体像素值，用于给阴影添加模糊效果，不允许为负值。值越大阴影越模糊，默认值为0
spread 可选，必须为具体像素值，正值使阴影扩展，负值使阴影收缩，默认为0
Chrome, Safari 和 Opera等浏览器不支持该参数，添加了可能导致阴影不渲染*
color 可选，指定阴影的颜色，不指定则根据浏览器来定（一般是黑色）

多滤镜

可以同时应用多个滤镜，多滤镜函数之间用空格分隔。
