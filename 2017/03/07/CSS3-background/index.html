CSS3 背景

前端CSS3
  • CSS3

CSS3包含了几个新增的背景属性，使我们能更有效地控制元素的背景：

  • background-size
  • background-origin
  • background-clip
  • CSS3 Backgrounds

CSS3背景图片尺寸

CSS3中的background-size属性可以用来指定背景图片的尺寸；在CSS3之前，背景图片的尺寸是图片的实际大小，CSS3允许指定背景图片的大小。

图片尺寸可以指定为实际尺寸（像素值）、百分比，也可以使用contain和cover两个关键字。

background-size指定为实际像素或百分比时，可接受两个值，分别指定背景图片宽和高的尺寸；如果只接受一个值，该值用于图片的宽，图片的高等比例缩放。

contain和cover关键字的意思：

  • contain 缩放背景图片到最大尺寸（宽和高必须保证在内容区域内），也就是说依赖于背景图片和设置区域的比例，背景图片有可能覆盖不到某些区域。
  • cover 缩放背景图片以至于图片覆盖到整个内容区域（图片宽和高都要等于或超过该区域），也就是说背景图片某些部分有可能不会显示出来。

CSS3 background-origin 属性

CSS3的background-origin属性指定背景图片定位在哪，该属性可以设置三个值：

  • border-box 背景图片从边框的左上角开始
  • padding-box （默认值）背景图片从填充（padding）边缘的左上角开始
  • content-box 背景图片从内容（排除padding）的左上角开始

CSS3 background-clip 属性

CSS3的background-clip属性用来指定背景色的绘制区域，与background-origin类似，该属性同样可以设置三个值，意义也基本一样：

  • border-box （默认值）背景绘制到边框的外边缘
  • padding-box 背景绘制到填充（padding）区域的外边缘
  • content-box 背景绘制在内容（排除padding）区域内

CSS3背景属性

属性 描述
background 在一条语句中设置所有背景属性的缩写属性
background-clip 指定背景色的绘制区域
background-image 为某元素指定一个或多个背景图片
background-origin 指定背景图片定位在哪
background-size 指定背景图片的尺寸大小
